About CS API

CS API is a personal hobby project providing a read-only REST API for Counter-Strike match, team, and player data. It is not affiliated with HLTV.org, Valve, or any tournament organizer.

Data

Data is sourced from publicly available information and refreshed once daily. No guarantees are made about accuracy or completeness.
